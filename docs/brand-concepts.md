# craftgarden ブランドコンセプト 5案

> 作成日: 2026-02-22
> 目的: craftgarden.studio のリブランディング方向性の検討

---

## 背景

- **craftgarden** = craft（手仕事、職人技）+ garden（庭、育む場所）
- ソロビルダー（個人開発者）のプロダクトスタジオ
- AIと情熱でデジタルプロダクトをつくる工房
- 10個のプロダクト（Webアプリ、ツール、コンテンツ）を公開中
- 現行: ダークテーマ（`#0a0e1a`）のテック寄りデザイン

## 案1: Botanical Code — 植物とコードの共生

### コンセプト名

**Botanical Code** — 植物的コード

### キーフレーズ

> Where code takes root and ideas bloom.
> （コードが根を張り、アイデアが花開く場所。）

### ムードキーワード

`有機的精密` `深い森` `静謐` `成長の気配` `露に濡れた葉` `夜のボタニカルガーデン` `生命回路`

### カラーパレット

| 役割 | 色名 | HEX | 用途 |
|------|------|-----|------|
| メイン1 | Deep Soil | `#0C1117` | 背景 — 豊かな土のような漆黒 |
| メイン2 | Moss Dark | `#1A2B23` | カード・セカンダリ面 — 苔むした深緑 |
| メイン3 | Pale Lichen | `#D4E0D7` | テキスト — 苔色がかった白 |
| アクセント1 | New Growth | `#34D399` | CTA・リンク — 新芽のグリーン |
| アクセント2 | Pollen Gold | `#FBBF24` | ハイライト — 花粉のゴールド |

### タイポグラフィの方向性

- **見出し**: ジオメトリック・サンセリフ（例: Inter, DM Sans）— 精密さと読みやすさを兼備
- **本文**: ヒューマニスト・サンセリフ — テックの硬さをやわらげる有機的なカーブ
- **コード・ロゴ**: モノスペース（JetBrains Mono）— 開発者としてのアイデンティティ維持
- **全体**: ミディアム〜ボールドウェイト中心。レタースペーシングはタイトめ

### ビジュアルモチーフ

- **回路基板×葉脈**: 回路パターンが葉脈に変化するグラデーション的モチーフ
- **成長リング**: プロダクトカードに年輪のような微細な同心円パターン
- **ルートライン**: セクション間をつなぐ根のような細いライン装飾
- **ドットグリッド**: 種子が整列したようなグリッドパターン（背景テクスチャ）

### UIの特徴

- **レイアウト**: 現行のシングルページ構成を踏襲。余白はやや広めに
- **アニメーション**: 控えめ — スクロールで要素が「芽生える」ようにフェードイン。カードホバーは根が伸びるような下方向のシャドウ拡張
- **インタラクション**: ホバー時にカードの縁が`New Growth`色でほのかに発光
- **特徴**: ダークベースを維持しつつ、無機質さを緑のアクセントで中和。現行デザインからの移行コストが最も低い

## 案2: Morning Garden — 朝の庭

### コンセプト名

**Morning Garden** — 朝の庭

### キーフレーズ

> Built at dawn, one product at a time.
> （夜明けとともにつくる、ひとつずつ。）

### ムードキーワード

`朝露` `やわらかな光` `温もり` `手仕事の誠実さ` `穏やかな自信` `白い朝` `新鮮`

### カラーパレット

| 役割 | 色名 | HEX | 用途 |
|------|------|-----|------|
| メイン1 | Morning Mist | `#FAFAF7` | 背景 — 朝霧のようなウォームホワイト |
| メイン2 | Linen | `#F0EDE6` | カード面 — リネンの質感 |
| メイン3 | Deep Bark | `#2C2418` | テキスト — 樹皮のようなダークブラウン |
| アクセント1 | Terracotta | `#C2704E` | CTA・リンク — テラコッタの温かみ |
| アクセント2 | Sage | `#7D9B76` | バッジ・サブアクセント — セージグリーン |

### タイポグラフィの方向性

- **見出し**: セリフ体（例: Playfair Display, Lora）— クラフト感と上品さ
- **本文**: 柔らかいサンセリフ（例: Source Sans 3）— 読みやすく温かい
- **ロゴ**: セミセリフまたはスラブセリフ — 手仕事の看板のような風合い
- **全体**: ライトからレギュラーウェイト。ゆったりとした行間（1.8〜2.0）

### ビジュアルモチーフ

- **手描き風ボーダー**: セクション区切りに手描きのような揺らぎのあるライン
- **テクスチャ**: 紙や木目を連想させる微細なノイズテクスチャ（CSSのみ）
- **植物シルエット**: ヘッダーやフッターに控えめなボタニカルシルエット（SVG）
- **格子**: 畑の畝を思わせる整然としたグリッドライン

### UIの特徴

- **レイアウト**: ワイドな余白。コンテンツエリアは狭め（max-width: 960px）で読みやすさ重視
- **アニメーション**: ほぼなし。ページ遷移のみクロスフェード。静かで落ち着いた印象
- **インタラクション**: ホバー時にカードが微かに浮く＋影が柔らかく広がる。テラコッタ色のアンダーライン出現
- **特徴**: ライトテーマへの大転換。温かみと誠実さで差別化。個人の「手仕事」感を最大化

### ポジション


## 案3: Neon Greenhouse — ネオン温室

### コンセプト名

**Neon Greenhouse** — ネオン温室

### キーフレーズ

> A digital greenhouse glowing with experiments.
> （実験が光る、デジタルの温室。）

### ムードキーワード

`サイバーボタニカル` `温室の蒸気` `ネオンの反射` `実験的` `大胆` `未来の植物園` `エレクトリックグリーン`

### カラーパレット

| 役割 | 色名 | HEX | 用途 |
|------|------|-----|------|
| メイン1 | Void Black | `#09090B` | 背景 — 深い闇 |
| メイン2 | Carbon | `#18181B` | カード面 — カーボン |
| メイン3 | Cloud White | `#FAFAFA` | テキスト — クリアホワイト |
| アクセント1 | Electric Mint | `#00FFB2` | 主アクセント — エレクトリックミント |
| アクセント2 | Hot Pink | `#FF3EAC` | 副アクセント — ホットピンク（花） |

### タイポグラフィの方向性

- **見出し**: ボールド・サンセリフ（例: Space Grotesk, Outfit）— 力強くモダン
- **本文**: ニュートラル・サンセリフ（例: Inter）— 高い可読性
- **ロゴ**: カスタムレタリング風のウェイトバリエーション。ネオン管のような細さ
- **全体**: エクストラボールドの見出し×ライトな本文のコントラスト。大きなサイズ差

### ビジュアルモチーフ

- **グロウエフェクト**: ネオンサインのようなテキストグロウ（text-shadow / box-shadow）
- **ガラス壁**: カードにグラスモーフィズム（backdrop-filter: blur + 半透明背景）
- **蒸気パーティクル**: 背景にゆっくり漂う微粒子アニメーション（CSS keyframes）
- **格子ガラス**: 温室のガラスパネルを連想させるグリッド装飾

### UIの特徴

- **レイアウト**: ベンドグリッド — カードが不均等なサイズで並ぶ実験的レイアウト
- **アニメーション**: リッチ — グロウのパルス、ホバーでの色変化、スクロール連動のパララックス。ただし`prefers-reduced-motion`完全対応
- **インタラクション**: カードホバーでネオンの色が変わる。カテゴリフィルターは光るタブ。マウスに追従する微かなグロウスポット
- **特徴**: 最も大胆で実験的。「遊び場」としての温室を体現。テック系ポートフォリオとして強い印象

## 案4: Wabi-Sabi Workshop — 侘び寂びの工房

### コンセプト名

**Wabi-Sabi Workshop** — 侘び寂びの工房

### キーフレーズ

> Imperfect, incomplete, and intentional.
> （不完全で、未完成で、意図的。）

### ムードキーワード

`侘び寂び` `余白の美` `素朴` `意図的な不完全` `静寂` `墨` `経年変化`

### カラーパレット

| 役割 | 色名 | HEX | 用途 |

| メイン1 | Washi | `#F5F1EB` | 背景 — 和紙のようなオフホワイト |
| メイン2 | Kitsune | `#E8E0D4` | カード面 — 狐色の淡い面 |
| メイン3 | Sumi | `#2D2926` | テキスト — 墨色 |
| アクセント1 | Bengara | `#8B4B3E` | リンク・CTA — 弁柄色（赤土） |
| アクセント2 | Wasurenagusa | `#6B8FA3` | サブアクセント — 勿忘草色（青灰） |

### タイポグラフィの方向性

- **見出し**: Noto Serif JP — 明朝体の凛とした美しさ。日本語の美しさを前面に
- **本文**: Noto Sans JP — ニュートラルで読みやすいゴシック
- **英文**: システムセリフ（Georgia, serif）— 日本語との調和
- **全体**: レギュラーウェイト中心。余白で語る。字間はデフォルト〜やや広め

### ビジュアルモチーフ

- **余白**: 最大のモチーフは「何もない空間」。情報を削ぎ落とす
- **墨のライン**: セクション区切りに毛筆の一筆を思わせる細い水平線
- **枯山水グリッド**: プロダクト一覧を枯山水の石の配置のように非対称に配置
- **印章**: ロゴに朱色の印章モチーフ（小さなアクセント）

### UIの特徴

- **レイアウト**: 非対称。意図的にグリッドを崩す。左寄せのテキストブロック＋右の大きな余白
- **アニメーション**: なし。完全に静的。状態変化のみ即座に反映（transition: 0ms）
- **インタラクション**: ホバーで弁柄色のアンダーラインが現れる。それ以外の装飾は排除
- **特徴**: 最もミニマル。和の美意識で唯一無二の個性。日本人ソロビルダーのアイデンティティを強調

## 推奨の視点

各案の選定は以下の問いで整理できる：

1. **現行デザインの延長線上で洗練したい** → 案1 Botanical Code
2. **温かみと誠実さで個人の手仕事を伝えたい** → 案2 Morning Garden
3. **テックポートフォリオとして強い印象を残したい** → 案3 Neon Greenhouse
4. **日本人ビルダーとして唯一無二の個性を打ち出したい** → 案4 Wabi-Sabi Workshop
5. **楽しさと実験精神を前面に出したい** → 案5 Playful Meadow

また、複数案のハイブリッドも有効。例えば：
- **案1 + 案4**: ダーク×和の融合（禅テック）
- **案2 + 案5**: ライトテーマ＋遊び心（温かいポップ）
- **案1 + 案3**: ダーク×グロウ（洗練されたネオンボタニカル）
